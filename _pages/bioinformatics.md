# Introduction to bioinformatics

An introduction to analyzing NGS data with bioinformatics including programs, workflows, educational videos, and relevant terminology.

## 1. Overview
### What is bioinformatics?

Bioinformatics is an interdisciplinary field that combines computer science and statistics to analyze and interpret biological datasets. Genomics is a specific branch of bioinformatics that focuses on analyzing DNA sequencing data to glean biologically relevant insights

## 2. Glossary of bioinformatics terms
  * **Alignment-** The process of computationally arranging sequencing reads against a reference genome.
  * **Basecalling-**The process of determining the sequence of nucleotides in a DNA sample during sequencing.
  * **Command line interface (CLI)-**A text-based user interface for interacting with a computer’s operating system by typing commands into a terminal window 
  * **Consensus sequence-**The representative DNA, RNA, or amino acid sequence that represents each nucleotide or amino acid that occurs most frequently at a given site.
  * **Contig-**Overlapping DNA segments that represent a contiguous region of a DNA sequence.
  * **De Novo Assembly-**The process of building a complete genome sequence from short, fragmented DNA sequences without relying on a pre-existing reference sequence.
  * **Depth-**The average number of times a particular nucleotide in a nucleic acid sequence is read during the sequencing process. Higher sequencing depth values correspond to higher confidence in individual base calls.
  * **FASTA-**Text-based file format for storing biological sequencing data (quality scores not included). 
  * **FASTQ-**Text-based file format for storing biological sequencing data and sequencing quality scores.
  * **Graphical user interface (GUI)-**A visual system for interacting with a computer’s operating system by using icons, menus, and windows instead of text-based commands.
  * **Library-**A collection of DNA or RNA fragments derived from a sample and prepared with specific adapter and index sequences to facilitate multiplex sequencing and subsequent demultiplexing.
  * **Quality score-**Also referred to as a ‘Q Score’ or ‘Phred Score’. A numerical value (Q) indicating the probability (p) that a given base call is correct: Q = -10log10p 

**Read-**The sequence of letters generated by a sequencing machine when it analyzes a fragment of DNA or RNA. 
**Reference Genome-**A representative nucleic acid sequence used as a standard for comparing and analyzing new DNA sequences.
**SAM/BAM/CRAM-**SAM/BAM/CRAM - Sequencing file formats used to store sequence data that has been aligned to a reference genome.
